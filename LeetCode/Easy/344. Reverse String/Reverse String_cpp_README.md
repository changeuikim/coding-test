# [Easy] [Reverse String](https://leetcode.com/problems/reverse-string) - 344

### 구분

알고리즘, 데이터구조

## C++ 문제풀이

```cpp
class Solution {
public:
    void reverseString(vector<char>& s) {
        reverse(s.begin(), s.end());
    }
};
```
